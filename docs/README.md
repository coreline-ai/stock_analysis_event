# DEEPSTOCK
## Research-Only Signal & Timing Engine

DEEPSTOCK는  
**실제 매매를 수행하지 않는 시장 시점 판단 및 리서치 엔진**입니다.

이 프로젝트의 목적은  
“자동으로 주문을 넣는 것”이 아니라  
**“지금이 진입을 고려할 시점인지 판단하고, 그 근거를 설명하는 것”**입니다.

---

## ✨ 핵심 개념

- ❌ 자동 매매 봇 아님
- ❌ 브로커 연동 없음
- ❌ 주문 실행 없음

- ✅ 소셜/뉴스 시그널 수집
- ✅ 시그널 스코어링 및 랭킹
- ✅ LLM 기반 리서치 분석
- ✅ **BUY_NOW / WATCH / AVOID 시점 판단**
- ✅ 근거·리스크·신뢰도 설명
- ✅ 데일리 리포트 및 히스토리
- ✅ 대시보드 / API 제공

> DEEPSTOCK는 **“시장 판단 엔진(Signal & Timing Engine)”**입니다.

---

## 🧠 무엇을 제공하나요?

DEEPSTOCK는 사용자가 다음 질문에 답을 얻도록 돕습니다:

- 지금 이 종목은 진입을 고려할 시점인가?
- 왜 지금인가?
- 어떤 전제가 필요하고, 어떤 리스크가 있는가?

판단 결과는 항상 다음 중 하나입니다:

- **BUY_NOW** : 지금 진입을 고려할 시점
- **WATCH** : 조건 충족 시 진입 가능
- **AVOID** : 리스크 또는 노이즈 구간

⚠️ 이 판단은 **어떤 경우에도 실제 주문을 실행하지 않습니다**.

---

## 🏗️ 시스템 개요

- 배포 환경: **Vercel**
- 실행 방식:
  - Serverless API Routes
  - Cron 기반 주기 실행
- 실행 모델:
  - 상시 실행 서버 없음
  - 요청 또는 스케줄 기반 실행
- 데이터 저장:
  - 시그널 / 판단 / 리포트 / 실행 이력 영속 저장

## 🔒 안전성 원칙 (중요)

DEEPSTOCK는 다음을 **강제적으로 보장**합니다:

- 실제 매매 실행 경로 없음
- 브로커 API 키 존재 시 시스템 부팅 실패
- 주문 실행 함수는 존재하더라도 항상 실패 처리
- 판단(Decision)은 기록되지만 실행(Execution)은 불가능

이 프로젝트는 **리서치 및 판단 전용**입니다.
